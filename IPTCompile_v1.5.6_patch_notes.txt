IPTCompile v1.5.6 - Resample Classification
Datum: 2026-07-22
Baslinje: IPTCompile v1.5.5 Review Focus

KORRIGERING
v1.5.5 klassificerade Resample endast via RESAMPLE/RESAMPLING i filnamnet och ignorerade exakt RES-prefix i Sample ID. Detta är korrigerat.

IMPLEMENTERADE REGLER
1. Filnamn klassificeras som Resample när det innehåller:
   - RES som tydlig filnamnskomponent, oavsett skiftläge
   - Resample
   - Resampling
   Exempel som matchar: LSP123_RES.csv, LSP123Res.csv, LSP123_Resample.csv, LSP123_Resampling.csv.
   Exempel som inte matchar: LSP123_RESULTS.csv, LSP123_Pressure.csv.

2. Sample ID klassificerar filen som Resample endast när första token före första understrecket är exakt RES.
   Exempel: RES_SP_N_001 -> Resample.

3. RES senare i Sample ID är inte en Resample-signal.
   Exempel: A14MAR27_RES_N_001 -> Primary-signal.

4. Vid två CSV-filer accepteras endast entydigt:
   - exakt en Primary
   - exakt en Resample

   Följande stoppas med tydligt fel:
   - två Primary
   - två Resample
   - blandade RES- och övriga prefix i samma CSV
   - okänd/otydlig roll där Primary + Resample inte kan verifieras

OPTIMERING
- Ett tydligt Resample-filnamn avgör rollen utan att CSV-innehållet behöver läsas för klassificering.
- När filnamnet inte avgör rollen läses CSV:n en gång för att kontrollera Sample ID-prefix.
- Det ordinarie staged CSV-context-flödet är oförändrat.

ÄNDRADE FILER
- Infrastructure/Csv.ps1
- Modules/Config.ps1
- ReleaseNotes.psd1
- Version.txt

VERIFIERING
- Syntetiska klassificeringsfall verifierade för RES/Res/Resample/Resampling.
- RESULTS och Pressure verifierade som icke-Resample via filnamn.
- Exakt RES-prefix, RES senare i Sample ID och blandade prefix verifierade.
- Pair-matris verifierad: endast Primary + Resample godkänns.
- PowerShell-struktur kontrollerad för ändrade filer.
- Sammanfogat BuildFlow-flöde har oförändrad och balanserad struktur.
- Excelmall, EPPlus, Open XML-bibliotek, ikon och övriga produktionsfiler är byte-identiska med v1.5.5.

BEGRÄNSNING
Full Windows PowerShell 5.1/WinForms-körning kunde inte utföras i Linuxmiljön. Funktionstesta därför minst:
- en Primary CSV utan Resample
- Primary + filnamn med RES
- Primary + filnamn med Resample/Resampling
- Primary + neutralfilnamn där Sample ID börjar med RES_
- två Primary
- två Resample
- CSV med blandade RES- och andra prefix
